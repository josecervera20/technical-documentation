Introduction

Computer networks enable communication between devices through data exchange.

They are fundamental in the operation of the internet and system interconnectivity.

The objective of this documentation is to provide an overview of computer networks, including their types, protocols, security, and essential tools.

Networking is essential for both personal and business operations.

It allows devices to share resources, information, and access to the internet.

Network Types

There are different types of networks based on their scope and functionality:

  • LAN (Local Area Network): Connects devices within a specific location, such as an office or home.
  • WAN (Wide Area Network): Covers large distances and connects multiple locations.
  • MAN (Metropolitan Area Network): Designed for urban areas, interconnecting several local networks.
  • WLAN (Wireless Local Area Network): Wireless networks that enable cable-free connections.

Each type of network has different use cases, from small offices to large-scale global networks.

Protocols

Communication protocols are rules that allow data transfer between devices in a network:

  • TCP/IP: A suite of protocols forming the foundation of the internet and networks.
  • HTTP/HTTPS: Protocols used for secure web browsing.
  • FTP: File transfer protocol between clients and servers.
  • DNS: System that translates domain names into IP addresses.
  • SMTP: Protocol for sending emails.

Example of an HTTP request:

GET /index.html HTTP/1.1

Example of a POST request:

POST /submit-form HTTP/1.1

Checking network connectivity using ping:

ping www.example.com
Security

Network security is crucial to protect data and prevent unauthorized access:

  • Data Encryption: Ensures the privacy of transmitted information.
  • Firewalls: Filter network traffic to block unauthorized access.
  • VPNs: Virtual private networks that secure remote connections.
  • Multi-Factor Authentication: Adds an extra layer of security by requiring multiple verification methods.
  • Intrusion Detection: Tools to monitor suspicious activities in the network.

Secure remote login using SSH:

ssh user@hostname
Tools

Various tools exist for managing and diagnosing computer networks:

  • Wireshark: Network traffic analyzer.
  • nmap: Tool for network scanning.
  • Netcat: Used for communication testing and diagnostics.
  • Ping and Traceroute: Utilities for checking connectivity and network routes.
  • SNMP: Protocol for network device monitoring and management.

Scanning a network for active devices:

nmap -sP 192.168.1.0/24

Pinging a public DNS server:

ping 8.8.8.8
